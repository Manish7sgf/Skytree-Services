```

         вҶ“

         вҶ“

         вҶ“

         вҶ“

         вҶ“
CSS calculates new --color-variables
         вҶ“
Browser repaints all elements
         вҶ“

Cards: Dark borders вҶ’ Light borders

         вҶ“

         вҶ“
```
